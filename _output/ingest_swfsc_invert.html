1 Overview

Source: NOAA CoastWatch ERDDAP

  • erdCalCOFIinvcnt — invertebrate counts
  • erdCalCOFIinvsiz — invertebrate sizes

Counts and sizes of invertebrates (primarily squids) captured in CalCOFI ichthyoplankton nets, processed by SWFSC. The two ERDDAP datasets share the same sampling metadata columns; invcnt adds count/abundance fields, invsiz adds a size measurement field.

This ingest creates two output tables:

  • invert_net — unique sampling events (xyt: time, location, net metadata)

  • invert — measurements in long format (count, standardized counts, size) with FK to invert_net and species identifiers

  • Provider: swfsc

  • Coverage: 1978–2023 (counts), 1980–2023 (sizes)

  • Spatial: Pacific coast, 0–51°N

11 Build cruise_key

The ERDDAP cruise column is YYYYMM format (e.g., 197804 = April 1978) and ship_code is the 2-letter ship key (e.g., “JD”). Combine with the ship table’s ship_nodc to create cruise_key in YYYY-MM-NODC format for matching to the ichthyo cruise hierarchy.

Code 
# add cruise_key to both source tables
for (tbl in c("invert_count", "invert_size")) {
  dbExecute(con, glue(
    "ALTER TABLE {tbl} ADD COLUMN IF NOT EXISTS cruise_key VARCHAR"))
  dbExecute(con, glue("
    UPDATE {tbl} AS ic
    SET cruise_key = CONCAT(
      LEFT(CAST(ic.cruise AS VARCHAR), 4), '-',
      SUBSTR(CAST(ic.cruise AS VARCHAR), 5, 2), '-',
      s.ship_nodc)
    FROM ship s
    WHERE s.ship_key = ic.ship_code"))
}

# report match rates
for (tbl in c("invert_count", "invert_size")) {
  n_total <- dbGetQuery(con, glue("SELECT COUNT(*) FROM {tbl}"))[[1]]
  n_match <- dbGetQuery(con, glue(
    "SELECT COUNT(*) FROM {tbl} WHERE cruise_key IS NOT NULL"))[[1]]
  n_cruise <- dbGetQuery(con, glue("
    SELECT COUNT(*) FROM {tbl}
    WHERE cruise_key IN (SELECT cruise_key FROM cruise)"))[[1]]
  cat(glue("{tbl}: {n_match}/{n_total} rows got cruise_key, ",
           "{n_cruise} matched to ichthyo cruise table"), "\n")
}

13 Create invert — measurements in long format

Pivot count and size measurements into a single long-format table, similar to how ichthyo stores egg/larva measurements. Each row is one measurement for one species at one sampling event.

Code 
# -- pivot invert_count: 3 measurement types per row ----
# join to invert_net to get invert_net_id
sql_count <- "
  SELECT
    n.invert_net_id,
    ic.scientific_name,
    ic.common_name,
    ic.itis_tsn,
    ic.calcofi_species_code,
    mt.measurement_type,
    mt.measurement_value
  FROM invert_count ic
  JOIN invert_net n
    ON  ic.cruise_key      = n.cruise_key
    AND ic.order_occupied   = n.order_occupied
    AND ic.tow_type         = n.tow_type
    AND ic.tow_number       = n.tow_number
    AND ic.net_location IS NOT DISTINCT FROM n.net_location
  CROSS JOIN LATERAL (VALUES
    ('count',           CAST(ic.invert_count  AS DOUBLE)),
    ('count_per_10m2',  ic.inverts_10m2),
    ('count_per_100m3', ic.inverts_100m3)
  ) AS mt(measurement_type, measurement_value)
  WHERE ic.cruise_key IS NOT NULL
    AND mt.measurement_value IS NOT NULL"

# -- pivot invert_size: size measurement ----
sql_size <- "
  SELECT
    n.invert_net_id,
    isz.scientific_name,
    isz.common_name,
    isz.itis_tsn,
    isz.calcofi_species_code,
    'size' AS measurement_type,
    isz.invert_size AS measurement_value
  FROM invert_size isz
  JOIN invert_net n
    ON  isz.cruise_key      = n.cruise_key
    AND isz.order_occupied   = n.order_occupied
    AND isz.tow_type         = n.tow_type
    AND isz.tow_number       = n.tow_number
    AND isz.net_location IS NOT DISTINCT FROM n.net_location
  WHERE isz.cruise_key IS NOT NULL
    AND isz.invert_size IS NOT NULL"

# combine and assign sequential id
dbExecute(con, glue("
  CREATE OR REPLACE TABLE invert AS
  SELECT
    ROW_NUMBER() OVER (
      ORDER BY invert_net_id, scientific_name, measurement_type, measurement_value
    ) AS invert_id,
    *
  FROM (
    {sql_count}
    UNION ALL
    {sql_size}
  ) sub"))

## Overview

**Source**: NOAA CoastWatch ERDDAP

- [erdCalCOFIinvcnt](https://coastwatch.pfeg.noaa.gov/erddap/tabledap/erdCalCOFIinvcnt.html) — invertebrate counts
- [erdCalCOFIinvsiz](https://coastwatch.pfeg.noaa.gov/erddap/tabledap/erdCalCOFIinvsiz.html) — invertebrate sizes

Counts and sizes of invertebrates (primarily squids) captured in CalCOFI
ichthyoplankton nets, processed by SWFSC. The two ERDDAP datasets share
the same sampling metadata columns; `invcnt` adds count/abundance fields,
`invsiz` adds a size measurement field.

This ingest creates two output tables:

- **`invert_net`** — unique sampling events (xyt: time, location, net metadata)
- **`invert`** — measurements in long format (count, standardized counts, size)
  with FK to `invert_net` and species identifiers

- **Provider**: `swfsc`
- **Coverage**: 1978–2023 (counts), 1980–2023 (sizes)
- **Spatial**: Pacific coast, 0–51°N

```{r}
    ic.itis_tsn,
    ic.calcofi_species_code,
    mt.measurement_type,
    mt.measurement_value
  FROM invert_count ic
  JOIN invert_net n
    ON  ic.cruise_key      = n.cruise_key
    AND ic.order_occupied   = n.order_occupied
    AND ic.tow_type         = n.tow_type
    AND ic.tow_number       = n.tow_number
    AND ic.net_location IS NOT DISTINCT FROM n.net_location
  CROSS JOIN LATERAL (VALUES
    ('count',           CAST(ic.invert_count  AS DOUBLE)),
    ('count_per_10m2',  ic.inverts_10m2),
    ('count_per_100m3', ic.inverts_100m3)
  ) AS mt(measurement_type, measurement_value)
  WHERE ic.cruise_key IS NOT NULL
    AND mt.measurement_value IS NOT NULL"

# -- pivot invert_size: size measurement ----
sql_size <- "
  SELECT
    n.invert_net_id,
    isz.scientific_name,
    isz.common_name,
    isz.itis_tsn,
    isz.calcofi_species_code,
    'size' AS measurement_type,
    isz.invert_size AS measurement_value
  FROM invert_size isz
  JOIN invert_net n
    ON  isz.cruise_key      = n.cruise_key
    AND isz.order_occupied   = n.order_occupied
    AND isz.tow_type         = n.tow_type
    AND isz.tow_number       = n.tow_number
    AND isz.net_location IS NOT DISTINCT FROM n.net_location
  WHERE isz.cruise_key IS NOT NULL
    AND isz.invert_size IS NOT NULL"

# combine and assign sequential id
dbExecute(con, glue("
  CREATE OR REPLACE TABLE invert AS
  SELECT
    ROW_NUMBER() OVER (
      ORDER BY invert_net_id, scientific_name, measurement_type, measurement_value
    ) AS invert_id,
    *
  FROM (
    {sql_count}
    UNION ALL
    {sql_size}
  ) sub"))

n_invert <- dbGetQuery(con, "SELECT COUNT(*) FROM invert")[[1]]
n_by_type <- dbGetQuery(con, "
  SELECT measurement_type, COUNT(*) AS n
  FROM invert
  GROUP BY measurement_type
  ORDER BY measurement_type")

cat(glue("invert: {format(n_invert, big.mark = ',')} total measurement rows"), "\n")
n_by_type |>
  datatable(caption = "Measurement rows by type")
```
